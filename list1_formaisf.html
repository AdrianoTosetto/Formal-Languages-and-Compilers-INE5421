Prova por construção(Gramática):

Sejam G1 = (V1, T1, P1, S1) a gramática da linguagem L1 e G2 = (V2, T2, P2, S2) a gramática para a linguagem 2. Agora construa uma gramática G3 = (V3, T3, P3, S3). Em P3 existe uma produção do tipo S3 -> S1 | S2

Concatenação

Prova por construção(Automato):

Pegue os dois automatos. Nos estados de aceitação do primeiro automato, adicione transições para o estado inicial do segundo automato.

Prova por construção(Gramática):

Para cada produção do tipo N -> x, em que N é um símbolo não terminal da primeira linguagem e x é um símbolo terminal da primeira linguagem, substitua-a por uma produção do tipo N -> xS, em que S é o símbolo inicial da segunda linguagem. Se ε pertencer à segunda linguagem, apenas adicione a produção anterior, sem substituir.

União

Prova por construção(Automato):

Pegue os dois automatos finitos e adicione um novo estado inicial. Nele, parta de ε para os antigos estados iniciais de cada automato. Como todo automato não determinístico pode ser convertido para um automato determinístico, então a união de suas linguagens regulares também será regular.

Para linguagens não regulares: Adicione um novo símbolo inicial S' que vai para S'S| S | ε. Isso é valido para todas as linguagens menores que o tipo 3, mas também é necessário.

Para linguagens regulares: Se ela for regular, para cada produção que gere uma sentença (que gera apenas um símbolo terminal), adicione S para manter a linguagem regular. Se existe S que gera ε, então essa produção deve ser removida.
